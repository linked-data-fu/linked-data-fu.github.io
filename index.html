Linked Data-Fu

An engine for a Linked Data processing/scripting language.

Linked Data-Fu is a processing/scripting language for Linked Data based on (production) rules in Notation3 syntax. The rule engine evaluating Linked Data-Fu programs supports large-scale data integration (including reasoning with ontologies) and includes the data manipulation protocol (such as high-frequency updates). Linked Data-Fu allows for the writing of succint specifications that operate over a Read/Write Linked Data abstraction. Linked Data-Fu is an end-to-end data processing system that can be used in data integration and system interoperation scenarios.
